# EasyBits

Sell digital assets with zero friction.

[easybits.cloud](https://www.easybits.cloud)

## Stack

React Router v7 · Prisma (MongoDB) · Stripe · Fly.io

## Getting started

```bash
npx prisma generate
npm run dev
```
